GENERAL



  1. What is eAuction System ?

    2. -> An online auction that takes place in real time. It gives suppliers the opportunity to bid against each other to improve their offers.

  2. Can I access my eAuction Account from a different client system ?

    3. -> Yes, user can access his account from anywhere with the above mentioned configuration (as in S. no. 3) using the DSC which is mapped with the account.

  3. What is the default Date and Time format used in the System ?

    4. -> The Default date time format is Indian Standard Time (IST), which is (GMT +5:30). The format used is DD-Mon-YYYY HH:MM AM/PM. This standard convention is followed in all the pages.

  4. Many documents have to be uploaded as PDF document. Sometimes, it is seen that Autocad drawings are required to be uploaded. Can I have some software for creating / viewing these documents ?

    5. -> In the Home Page, under the downloads link, links for the software that are required for working with the application are given. These links point to the Open Source software and hence no licensing is required for using any of these software. Few of the links include Open Office, Adobe Reader, PDF Creator, DWF Viewer etc.

  5. I have uploaded the documents after digitally signing all the documents. Should I submit paper documents ink signed and submit to the AIA office ?

    6. -> No. As per the IT ACT 2000, the digitally signed document is as good as paper document for all legal purposes. Hence, there is no need to submit all the documents physically. However, when the physical documents are requested for cross checking purposes, the same need to be produced for verification. However, In the case of offline payment instruments, the physical documents need to be submitted before the bid opening event.

  6. Can Auction be conducted in Foreign Currencies ?

    7. -> Portal facilities the Auction process to be conducted in INR only.

  7. Can Auction be conducted in Foreign Currencies ?
